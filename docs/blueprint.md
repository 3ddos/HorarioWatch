# **App Name**: HorarioWatch

## Core Features:

- Gmail Account Integration: Securely connect to a specified Gmail account to monitor incoming emails.
- Email Filtering Configuration: Allow users to define specific criteria (e.g., sender, subject line) to identify target emails containing work schedules.
- PDF Attachment Retrieval: Automatically detect and download PDF attachments from emails matching the configured criteria.
- AI-Powered Schedule Extraction: Utilize an AI tool to intelligently parse PDF content, identify the structure of work schedules, and extract working days and hours for a specific person.
- Target Person Identification: Enable users to input the name of the person whose schedule needs to be extracted from the PDF.
- Schedule Data Storage: Store the extracted work days and times for the target person in a structured database.
- Processed Schedule Overview: Provide a simple interface to view a log of processed emails and the extracted schedule data.

## Style Guidelines:

- Primary color: A mature and trustworthy blue (#4564A1) to convey organization and reliability, fitting for data management.
- Background color: A very light, subtly cool off-white (#ECEFF3) for a clean, uncluttered interface that enhances readability.
- Accent color: A clear and crisp cyan (#66B8DB) to highlight interactive elements and provide visual interest while maintaining professionalism.
- Body and headline font: 'Inter', a neutral and highly readable sans-serif, suitable for presenting data and system information clearly across the application.
- Employ a set of simple, professional, and clear line icons that communicate actions and status effectively without distracting from the content.
- Adopt a clean, minimalist layout with generous whitespace to ensure easy navigation and scannability of schedules and configurations.
- Incorporate subtle, quick animations for state changes and data processing feedback, enhancing user experience without being intrusive.
